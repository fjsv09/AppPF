## Available Skills

The following skills are installed in `.gemini/skills/`:

- **brainstorming** - Design refinement and spec generation
- **systematic-debugging** - 4-phase root cause analysis
- **test-driven-development** - RED-GREEN-REFACTOR cycle
- **writing-plans** - Implementation planning
- **executing-plans** - Batch task execution
- **subagent-driven-development** - Task execution with review
- **finishing-a-development-branch** - Cleanup and merge workflow
- **requesting-code-review** - Pre-review verification
- **receiving-code-review** - Responding to feedback
- **using-git-worktrees** - Isolated development environments
- **verification-before-completion** - Final fix verification
- **writing-skills** - Creating new skills

## Usage

To use a skill, use the `view_file` tool to read the `SKILL.md` file in the corresponding skill directory under `.gemini/skills/`.
Follow the instructions in `using-superpowers/SKILL.md` for guidance on when to invoke skills.
